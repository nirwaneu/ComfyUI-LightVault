# ComfyUI-LightVault

A lightweight, zero-dependency, and ultra-fast media gallery & workflow inspector extension for ComfyUI.

## 🚀 Features
- **Lightweight & Fast:** Built natively without heavy Python dependencies or databases.
- **Separate Tabs:** Easily toggle between `Output` and `Input` directories.
- **Media Support:** Displays both Images and Videos with hover-play preview.
- **Auto Model Tagging:** Automatically detects models (FLUX, WAN, Anima, Pony, Qwen, SDXL, etc.).
- **Workflow Inspector:** Load workflows directly into a **New Tab** without overwriting your active canvas.
- **Details Modal:** Copy positive/negative prompts easily and check metadata on the fly.

## 📦 Installation
Navigate to your ComfyUI `custom_nodes` directory and clone this repository:

cd ComfyUI/custom_nodes
git clone [https://github.com/nirwaneu/ComfyUI-LightVault.git](https://github.com/nirwaneu/ComfyUI-LightVault.git)
